Сравнение Playwright и Cypress

Playwright:

  • + Запускается через jest
  • + можно выбрать браузер и устройство для тестирования
  • — Излишний синтаксис (каждый раз async и await)

Cypress:

  • + Удобный ui при запуске тестов через cypress open
  • + сам записывает видео и скрины (для ошибочных тестов)
  • + документация лучше у cypress чем playwright
  • — Нельзя запустить через jest

Мне больше понравился playwright, потому более интуитивно понятные команды.

Тестирование с помощью скриншотов и видео

С помощью e2e тестов мы можем тестировать поведение всей системы и её реакцию на действия пользователя.
Тесты со скриншотами оказались сложноприменимыми в таком тривиальном приложении, но для проектов покруче с нормальным ui могут оказаться крайне полезными, особенно если адаптивное.
